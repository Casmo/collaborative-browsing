About This Demo

This page uses the Collaborative Browsing library, which creates a real-time shared browsing experience using PeerJS. Every visitor to the site can see where other visitors are and what they're looking at.

Features

  • See mouse cursors with name labels of other visitors in real-time
  • Each visitor gets a unique color
  • Cursors fade out when visitors are on different pages
  • Link badges show how many visitors are on each page
  • Click ripple animations on remote clicks
  • Scroll position indicators on the right edge
  • Text selection highlights from other visitors
  • Floating chat panel for real-time messaging
  • Optional sound effects
  • Completely peer-to-peer using WebRTC

How to Add to Your Website

Include the script and call CollaborativeBrowsing.start() with your desired options:

<script src="dist/collaborative-browsing.js"></script>
<script>
  CollaborativeBrowsing.start({
    userLabel: 'Alice',   // optional: set your display name
    sounds: false         // optional: enable sound effects
  });
</script>
